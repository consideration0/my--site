Python基础语法

1. 变量和数据类型

  • • 变量定义：使用等号(=)赋值
  • • 基本数据类型：整数(int)、浮点数(float)、字符串(str)、布尔值(bool)
  • • 示例：name = "Python"、age = 18、price = 9.99

2. 运算符

  • • 算术运算符：+、-、*、/、//、%、**
  • • 比较运算符：==、!=、>、<、>=、<=
  • • 逻辑运算符：and、or、not

3. 控制流

  • • if语句：条件判断
  • • for循环：遍历序列
  • • while循环：条件循环

4. 函数

  • • 使用def关键字定义函数
  • • 函数参数和返回值
  • • 示例：def add(a, b): return a + b

5. 列表和字典

  • • 列表：有序集合，使用方括号[]
  • • 字典：键值对集合，使用花括号{}
  • • 示例：list = [1, 2, 3]、dict = {"name": "Python", "version": 3.8}

6. 字符串操作

  • • 字符串拼接：使用+运算符
  • • 字符串格式化：使用format()方法或f-string
  • • 示例：f"Hello, {name}!"
